# Презентация: господдержка Desper Quartier 2026

## Описание

HTML-презентация на основе отчета `docs/financing/2026-02-21_government_programs_subsidies_report.md`.
Слайды отражают ключевые программы субсидий, льготного кредитования и налоговых инструментов для проекта D22 Projekt GmbH.

## Features

- Минималистичный визуальный стиль, вдохновленный `pencil.dev`
- Полностью статичный формат: открывается напрямую в браузере
- Навигация кнопками и клавиатурой (стрелки, Home, End)
- Прогресс-бар и счетчик слайдов
- Контент вынесен в `data/slides.js`, логика в `js/app.js`, стили в `css/styles.css`

## Структура

- `index.html` — оболочка презентации
- `css/styles.css` — визуальная тема и адаптив
- `data/slides.js` — данные слайдов
- `js/app.js` — рендеринг и навигация
